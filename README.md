# Homework 2 - LLM Workflow Prototype

## Business Workflow
This project prototypes a business workflow for drafting sales follow-up emails after client meetings.

## User
The main user is a sales representative who needs to send follow-up emails quickly.

## Input
Meeting notes including client name, company, discussion, and next steps.

## Output
A professional follow-up email.

## Why This Task Is Valuable
This task saves time and improves consistency in business communication.
